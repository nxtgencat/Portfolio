## Overview

This portfolio was built using **Once UI for Next.js**, designed to highlight projects, skills, and achievements effectively. It leverages modern web technologies to deliver a seamless user experience across all devices, with an emphasis on timeless design and functionality.

## Key Features

- **Once UI Integration:** Utilizes all tokens, components, and features of Once UI.
- **SEO Automation:** Automatic Open Graph and X image generation with `next/og`.
- **Schema & Metadata:** Automatic schema and metadata generation based on content files.
- **Responsive Design:** Optimized layout for all screen sizes, ensuring usability and accessibility.
- **Customizability:** Endless customization through data attributes, catering to diverse needs.
- **Content Management:** Enable or disable sections (blog, work, gallery, about/CV) and conditionally render sections based on content files.
- **Localization Support:** Fully integrated with `next-intl` for seamless multi-language functionality.
- **Password Protection:** Protect specific URLs with password-based authentication.

## Technologies Used

- **Frontend:** Next.js, Once UI
- **Localization:** next-intl library
- **Image Generation:** next/og
- **Design:** CSS, responsive layouts, and customizable data attributes
- **Content Management:** Dynamic content rendering with Markdown

## Challenges and Learnings

- **SEO Optimization:** Implementing automatic schema and Open Graph generation required a thorough understanding of metadata best practices.
- **Localization:** Integrating `next-intl` for multilingual support highlighted the importance of structured and reusable translations.
- **Customization:** Fine-tuning Once UI components to align with specific design requirements taught valuable lessons in component-based design.
- **Responsive Design:** Ensuring consistent performance across devices emphasized the importance of mobile-first development.

## Outcome

The portfolio successfully showcases projects and skills in a professional and visually appealing manner. It demonstrates expertise in Next.js and Once UI, while offering flexibility and scalability for future updates. This project emphasizes my ability to deliver modern, feature-rich web applications optimized for both users and developers.
